🚀 **About the Project**
This is a simple yet engaging Number Guessing Game where the player tries to guess a random number between 1 and 100. The game features:

A clean and intuitive GUI with real-time feedback.

Hints system to help you narrow down the number.

Score tracking based on how quickly you guess right.

Dark mode toggle for comfortable gameplay day or night.

Limited attempts and hints to keep the challenge alive.

Option to restart anytime and try for a better score.

Built entirely in Java Swing, this game is a great example of event-driven programming and GUI design.



🎮 **How to Play**
Enter your guess (1 to 100) in the text box.

Click Guess to submit your attempt.

The game will guide you with hints like "Too high", "Too low", or "Very close!".

Use the Hint button (max 3 times) for clues like even/odd, greater than 50, or divisibility.

You have 5 attempts to guess the number correctly.

Your score depends on how fast you guess the number.

Toggle Dark Mode for a different theme.

Click Restart to play again anytime.



✨ **Features**
Interactive GUI: Easy-to-use interface with responsive buttons and input fields.

Hints system: Up to 3 hints per game to strategically help you guess.

Dark Mode: Switch between light and dark themes on the fly.

Score Calculation: Encourages faster guessing by rewarding points.

Input Validation: Ensures only numbers between 1 and 100 are accepted.

Replayability: Restart button lets you enjoy infinite rounds.
